\doxysubsection{Detailed Description}
Takes a JSON formatted spec file in the constructor, parses it and extracts the relevant fields. 

The JSON should be in our custom \textquotesingle{}dobby\textquotesingle{} container format, this includes extra custom fields for things like /etc files, plugin configurations and so forth.

It\textquotesingle{}s main purpose is to convert the dobby spec file to a OCI conformant JSON file.
