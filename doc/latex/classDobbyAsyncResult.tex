\doxysubsection{Detailed Description}
Result object for async and deferred results. 

The behaviour of this object is different depending on how it was created; for Dobby\+Async objects, the get\+Results() method will wait for the function to complete in a separate thread before returning the results. For Dobby\+Deferred objects, the function is executed in the current thread when the get\+Results() method is called.
