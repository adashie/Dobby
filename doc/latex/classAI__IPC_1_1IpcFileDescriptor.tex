{\footnotesize\ttfamily int Ipc\+File\+Descriptor\+::fd (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption}) const}

Returns the Unix file descriptor contained by this DBus\+File\+Descriptor object. An invalid file descriptor is represented by the value -\/1.

Note that the file descriptor returned by this function is owned by the \mbox{\hyperlink{classAI__IPC_1_1IpcFileDescriptor}{Ipc\+File\+Descriptor}} object and must not be stored past the lifetime of this object. It is ok to use it while this object is valid, but if one wants to store it for longer use you should use the Ipc\+File\+Descriptor\+::dup() function.
