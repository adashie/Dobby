Scans the given path for any shared objects that implement the plugin entry points. 

This calls dlopen() on all the executable files in the given path (although it doesn\textquotesingle{}t recurse into subdirs), if the file has symbols create\+IDobby\+Rdk\+Plugin and destroy\+IDobby\+Rdk\+Plugin then it\textquotesingle{}s deemed to be a \textquotesingle{}rdk\textquotesingle{} plugin.

If loaded successfully the plugins are stored in an internal map, keyed off the plugin name.

\begin{DoxyReturn}{Returns}
False if unable to open the given directory, true otherwise. 
\end{DoxyReturn}
