Constructs an IPv6 address to be used by \mbox{\hyperlink{classDobby}{Dobby}}. 

Takes an IPv4 address type and merges it into IN6\+ADDR\+\_\+\+BASE to make a unique IPv6 address.

The final address will be 2080\+:d0bb\+:1e\+::nnnn\+:nnnn, where the \char`\"{}n\char`\"{}s will be replaced by the IPv4 address binary.
